# Instructions

## Prisma setup
```
npm run prisma:up
npm run prisma:deploy
```

## Prisma DB wipe
```
npm run prisma:reset
```

## Prisma seed DB
```
npm run prisma:seed
```

## Getting schema
```
npm i -g graphql-cli
graphql get-schema --project database
graphql codegen --project database
```

## Running project
`npm run start`
